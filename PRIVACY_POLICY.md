# HumanScore Privacy Policy

Last updated: March 5, 2026

## Overview

HumanScore is a Chrome extension that estimates AI-likely writing signals on LinkedIn pages.  
All analysis runs locally in your browser.

## Data Collection

HumanScore does **not** collect, sell, or share personal data.

HumanScore does **not**:

- send LinkedIn content to external servers
- use analytics or tracking tools
- use advertising SDKs
- collect browsing history outside LinkedIn

## Data Storage

HumanScore stores only user settings in `chrome.storage`:

- score threshold
- highlight on/off preference
- scan mode preference
- custom marker phrases entered by the user

These settings are used only to operate the extension features.

## Permissions Use

- `storage`: save and load extension settings
- `https://www.linkedin.com/*` host access: analyze visible LinkedIn text and render badges/highlights on LinkedIn pages only
- `activeTab` (if present): run user-triggered popup actions on the current tab

## Remote Code / External Services

HumanScore does not execute remote code.  
HumanScore does not call external APIs for scoring.

## Data Retention

No user content is transmitted or retained by external systems because processing is local-only.

## Contact

For privacy questions, contact the extension publisher through the Chrome Web Store listing.
